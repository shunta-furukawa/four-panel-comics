# ネタ帳 / バックログ

「これいつか 1 話にできそう」を貯めておく場所（1 話 = 4 シート × 4 コマ）。

- とにかく **書きとめる** ことを優先（クオリティは後で）
- 内容が膨らんできたら、`episodes/_template.md` をコピーしてエピソード化
- GitHub Issue で管理してもよい（`idea` ラベル運用）

| ファイル                     | 内容                            |
| ---------------------------- | ------------------------------- |
| [`backlog.md`](./backlog.md) | テキストで貯めていくネタ帳本体  |

## 運用のコツ

- 1 行で書く。1 ネタ 1 行
- カテゴリ（タグ）を頭に付ける: `[デバッグ]`, `[ミカ]`, `[AI]` など
- 取り消し線でアーカイブ可（履歴は Git に残るので消さなくてもよい）
