# シェイプ・ライン

親しみやすい丸みと太い輪郭で、読みやすく、記憶に残るビジュアルに。

## 基本シェイプ

| シェイプ        | 用途                                        |
| --------------- | ------------------------------------------- |
| 丸・円形        | キャラクターアイコン、吹き出し              |
| 角丸（大きめ）  | 各セクション・コマのコンテナ                |
| シンプルな線    | 背景・装飾線                                |
| 太めのアウトライン | キャラクターの輪郭                       |

## 線の太さ

- キャラ輪郭: 太め（4 〜 6px 相当）
- 内側のディテール: 中（2 〜 3px 相当）
- 装飾線: 細（1 〜 2px 相当）

## 角丸

- パネル: 大きめの角丸（16 〜 24px 相当）
- 吹き出し: 小〜中の角丸 + ぷっくりした見た目
