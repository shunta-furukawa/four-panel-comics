# リズム設計（4 シート × 4 コマ）

> 1 話 = 4 シート × 4 コマ = 全 16 コマ。
> シートで大きな起承転結、シート内で小さな起承転結。読み終わって余韻が残る構成を目指す。

## 二重の起承転結

| 単位 | 起 | 承 | 転 | 結 |
| ---- | -- | -- | -- | -- |
| **シート全体（4 枚）** | 状況の提示 | 展開・問題発生 | 転換・最大の盛り上がり | 落としと余韻 |
| **各シート内（4 コマ）** | フリ | 展開 | ズラし | 落とし |

シート単位で大ストーリー、シート内で小ストーリー。X で 1 枚ずつスワイプして読まれることを前提に、**各シート単独でも 1 つの 4 コマとして成立する** ように作る。

## シートの役割

| シート | 大役割     | 描くこと                                            | NG パターン                              |
| ----- | -------- | --------------------------------------------------- | ---------------------------------------- |
| 1     | **起**   | 場・キャラ・前提を立ち上げる                         | 情報を詰め込みすぎる                     |
| 2     | **承**   | テーマや問題を膨らませる                              | シート 1 と同じ温度で進めてしまう        |
| 3     | **転**   | ひねり / 反転 / テンションのピーク                   | 最終オチをここで出してしまう             |
| 4     | **結**   | 全体オチ + 余韻                                      | 説明セリフでオチを台無しにする           |

## 各シート内の役割（小コマ単位）

| コマ | 役割    | 中身                              |
| ---- | ------- | --------------------------------- |
| N-1  | フリ    | そのシートのテーマを示す           |
| N-2  | 展開    | 状況を一段進める                   |
| N-3  | ズラし  | テンションが変わる小さなひねり     |
| N-4  | 落とし  | そのシートの小オチ + 次シートへの引き |

> シート 4 の最後（4-4）は **シートの落としと全体のオチを兼ねる**。

## チェックリスト（投稿前）

### シート全体

- [ ] シート 1 で **誰が・どこで・何をしてるか** が分かる
- [ ] シート 2 で **テーマや問題** がはっきりする
- [ ] シート 3 で **温度（テンション）が変化** している
- [ ] シート 4 で **全体オチが立っていて余韻も残る**（説明していない）

### 各シート内

- [ ] N-3 で温度の変化を作っている
- [ ] N-4 で次シートへの引き / そのシート単独のオチが成立している
- [ ] セリフを音読してテンポが悪くないか
- [ ] 各シートが画像 1 枚として X で読まれても破綻しないか

### バグまる回

- [ ] バグまるが意味あり気な仕草をするコマが入っている（ただし最低限）
- [ ] 真相を断定する描写は避けている

## オチのバリエーション

- **意外な結末**: 想像と違う方向に終わる
- **発見**: 主人公が何かに気づく
- **空回り**: 結果として何も解決していないが面白い
- **無言オチ**: セリフなしで絵だけで落とす
- **メタオチ**: 漫画というフォーマットや読者を意識した落とし方

## 参考: 作りやすいテンプレ

シート単位で次のような大流れを置くと組みやすい:

- "あるある提示 → 共感 → ひねり → 想定外のオチ"
- "問題発生 → 解決っぽいアイデア → 試す → 別の問題が発生"
- "ひらめき → 実装 → 動いた！ → バグまる出現"
