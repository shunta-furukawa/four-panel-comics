# カラーパレット

ベースはモノトーン、アクセントにネオンピンクでワクワク感を演出。
シアンはデジタル感・テクノロジーを表す補助色。

| 役割              | 名前         | HEX       | 用途                                       |
| ----------------- | ------------ | --------- | ------------------------------------------ |
| メインブラック    | Main Black   | `#0F0F10` | 背景・主線                                 |
| ダークグレー      | Dark Gray    | `#2A2A2E` | パネル・カード背景                         |
| ホワイト          | White        | `#FFFFFF` | テキスト・ハイライト                       |
| **ネオンピンク**  | Neon Pink    | `#FF2D7A` | **アクセント（タイトル・主要装飾）**       |
| シアンブルー（サブ）| Cyan Blue   | `#6EE7F7` | デジタル感・補助アクセント                 |

## 使い方

- **画面の 70%** を Main Black + Dark Gray
- **20%** を White
- **10%** を Neon Pink + Cyan Blue（アクセント）
- ネオンピンクの使いすぎ注意。"ここぞ" で効かせる

## NG パターン

- 彩度の高いオレンジ・緑・黄をメインで使わない
- グラデーションは使ってもよいが、トーンを揃える（ピンク → 淡ピンクなど）
