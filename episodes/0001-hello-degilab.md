# はじめてのデジラボ

## あらすじ

シリーズのキックオフ回。Shunta がデジラボでアイリスと一緒に
"今日からなにか面白いことを始めよう" と決意する話。
キャラの説明はせず、行動と会話で関係性を見せる。

## 起承転結

### 1 コマ目（起）— セットアップ
- **場所**: デジラボの執務スペース。モニター 3 枚、コーヒー、付箋
- **登場**: Shunta、アイリス
- **状況**: いつものデスクで Shunta がコーヒーをすすっている
- **セリフ**:
  - Shunta:「今日も平和なデジラボだなぁ」
  - アイリス:「Shunta さん、平和すぎてネタが足りていません」

### 2 コマ目（承）— 広がり・問題発生
- **状況**: ホワイトボードに付箋を貼り出す Shunta
- **セリフ**:
  - Shunta:「じゃあ毎日 4 コマ描こう。記録にもなるし」
  - アイリス:「素晴らしいアイデアです。続くかどうかは別として」

### 3 コマ目（転）— アイデア・試行錯誤
- **状況**: GitHub のリポジトリを開く Shunta。設定ファイルを書き始める
- **セリフ**:
  - Shunta:「設定もキャラもぜんぶ Git で管理する。これなら忘れない」
  - アイリス:「エンジニアらしい解決策ですね……」

### 4 コマ目（結）— オチ
- **状況**: 真夜中。Shunta が README を熱心に書いている。アイリスは呆れ顔
- **セリフ**:
  - Shunta:「README が一番楽しい！」
  - アイリス:「4 コマ、まだ 1 話も描けていません」
