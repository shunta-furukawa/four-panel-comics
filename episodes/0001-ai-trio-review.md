# AI 3 体に PR を見せたら

## あらすじ

ちょっと自信のない実装を PR にした Shunta が、AI トリオに同時にレビューを依頼する。
チャッピー君は全肯定、クロード君はダメ出しの嵐、ジェミナイ君は全く別の 5 案を提示。
3 体いてもむしろ困った Shunta は、結局自分で Merge ボタンを押すしかない。

## 構成

**4 シート × 4 コマ = 全 16 コマ**。X 投稿時はシートを 4 枚画像で添付。

- シート全体: 起 (PR を出す) → 承 (チャッピー全肯定) → 転 (クロード本気指摘) → 結 (ジェミナイ暴走 + 自分で決める)
- 各シート内: フリ → 展開 → ズラし → 落とし

---

## シート 1（起）— PR を出す

**シートの役割**: 場・キャラ・課題を立ち上げる
**画像**: `../assets/episodes/0001/sheet-1.png`

### 1-1（フリ）
- 状況: 深夜のデジラボ。Shunta がモニターに向かって作業。コーヒーマグから湯気
- セリフ:
  - Shunta:「よし、これで PR 出すか…」

### 1-2（展開）
- 状況: GitHub の "Create pull request" ボタンを押す瞬間
- セリフ:
  - Shunta:「（ポチッ）」

### 1-3（ズラし）
- 状況: PR 一覧画面を見て、ふと表情が曇る
- セリフ:
  - Shunta:「…でもちょっと自信ないな」

### 1-4（落とし）
- 状況: モニター 3 枚に向き直る。机の上にチャッピー君・クロード君・ジェミナイ君のアイコンが並んで光る
- セリフ:
  - Shunta:「3 人とも、見てくれる？」
  - チャッピー君（小窓）:「はい！」
  - クロード君（小窓）:「了解しました」
  - ジェミナイ君（小窓）:「いきまーす！」

## シート 2（承）— チャッピー君のレビュー

**シートの役割**: 期待 → 全肯定の波 → 違和感
**画像**: `../assets/episodes/0001/sheet-2.png`

### 2-1（フリ）
- 状況: チャッピー君のディスプレイが最初にピコンと光る
- セリフ:

- 状況: 拍手スタンプ・キラキラが画面いっぱいに流れる
- セリフ:

  - チャッピー君:「テストも丁寧で、命名にもセンスを感じます！」

### 2-3（ズラし）

- セリフ:
  - Shunta:「えっ、こんなに早く…？」

### 2-4（落とし）
- 状況: チャッピー君が超笑顔のまま Merge ボタンを指差す
- セリフ:
  - チャッピー君:「もう Merge して大丈夫ですよ！」
  - Shunta:「…ほんとに？」

## シート 3（転）— クロード君のレビュー

**シートの役割**: 整合性ガードによる本気の指摘
**画像**: `../assets/episodes/0001/sheet-3.png`

- 状況: クロード君のディスプレイが落ち着いた色で点灯。資料を手にした風
- セリフ:

### 3-3（ズラし）

- セリフ:
  - クロード君:「（以下 47 件）」
  - Shunta:（無言で天を仰ぐ）

### 3-4（落とし）
- 状況: 横でチャッピー君が驚いて両手を上げる。クロード君はクールに
- セリフ:
  - チャッピー君:「えっ、僕たち全肯定でしたよね！？」
  - クロード君:「肯定と整合性は別軸です」

## シート 4（結）— ジェミナイ君と、最後の決断

**シートの役割**: 全別案の暴走 → タクマ → 自分で決める
**画像**: `../assets/episodes/0001/sheet-4.png`

### 4-1（フリ）
- 状況: ジェミナイ君のディスプレイがキラキラ点灯。星屑をまとってご機嫌
- セリフ:
  - ジェミナイ君:「全部違う方向で 5 案考えてみました！」

### 4-2（展開）
- 状況: 画面に派手なビジュアル案 5 種が並ぶ。色も方向性もバラバラ
- セリフ:
  - ジェミナイ君:「アーキも UI も書き直していい雰囲気ですよ！」

### 4-3（ズラし）
- 状況: Shunta、頭を抱える。背景で AI トリオがそれぞれ違うトーン
- セリフ:
  - Shunta:「みんな全然違うじゃん…」

### 4-4（落とし）
- 状況: タクマがコーヒー片手に通りがかる
- セリフ:
  - タクマ:「で、結局どうすんの？」
  - Shunta:「…自分で決めます」

## 画像生成プロンプト（ChatGPT 用）

各シート 1 枚ずつ ChatGPT に投げる前提のプロンプト。
最初に **基本仕様** を投げてから、各シートのプロンプトを順に渡すと
スタイルが揃いやすい。

### 共通基本仕様（全シート）

```
[基本仕様 / 全シート共通]
- 縦長 1 枚の 4 コマ漫画レイアウト (推奨アスペクト比 3:4)
- 配色:
  - 背景: ほぼ黒 (#0F0F10) + ダークグレー (#2A2A2E)
  - アクセント: ネオンピンク (#FF2D7A)
  - サブ: シアン (#6EE7F7)
  - 文字: ホワイト (#FFFFFF)
- ヘッダ:
  - 左上: 「4コマ漫画」と書かれたピンクのタブ + "</>" 装飾
  - 中央: メインタイトル「AI 3 体に PR を見せたら」（大きめ）
  - サブタイトル「全肯定・全否定・全別案。困ったら自分で決めるしかない。」（赤角括弧でくくる）
  - 右上: ページインジケーター "N / 4" + 4 個の小さい丸（現在シートに当たる丸が塗り）
  - 右上にネオンピンクの電球アイコン
- 4 つの横長パネルを縦に並べる。各パネル左上にピンク丸の番号 (1, 2, 3, 4)
- フッタ:
  - 中央: 「AI が 3 体いても、最後に Merge ボタンを押すのは、いつだって自分。」
  - 左下: "</>" 装飾、右下: マウスカーソル装飾
- 絵柄: 親しみやすい丸み + 太めのアウトライン、漫画的だが落ち着いた色味
- セリフは日本語の漫画風吹き出し。読みやすいフォント

[キャラ設定（毎回同じ姿）]
- Shunta: 黒髪パーカー、頬がうっすらピンク、好奇心と少しの疲労が同居する表情
- チャッピー君: ピンクの六角形ロボ + ヘッドフォン風パーツ + 明るい目（ChatGPT/Codex 系）
- クロード君: 落ち着いたグレー角丸正方形ロボ + 暖色アクセント、資料持ち（Claude 系）
- ジェミナイ君: 水色ダイヤ型ロボ + ピンクの星屑、軽やかなシルエット（Gemini 系）
- タクマ: メガネ + 襟付きシャツ、コーヒーマグ常備、落ち着いたグレー基調
```
