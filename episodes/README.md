# エピソード索引

各話の脚本・ネーム・公開状況を管理します。

## 命名ルール

```
episodes/NNNN-slug.md
```

- `NNNN`: 4 桁の通し番号（例: `0001`, `0042`）
- `slug`: 内容を表す短い英小文字スラッグ（例: `hello-degilab`）

## status の意味

| status      | 意味                                       |
| ----------- | ------------------------------------------ |
| `draft`     | プロット段階。セリフ未確定                 |
| `inking`    | ネーム / 下書き作業中                      |
| `ready`     | 公開準備完了。投稿待ち                     |
| `published` | 公開済                                     |

## 一覧

| 番号   | タイトル                            | status | tags                 |
| ------ | ----------------------------------- | ------ | -------------------- |
| 0001   | はじめてのデジラボ                  | draft  | 自己紹介, 日常       |

## 新規エピソードの作り方

1. `_template.md` をコピー
   ```bash
   cp episodes/_template.md episodes/0002-your-slug.md
   ```
2. front matter を埋める
3. 起承転結の各コマ欄を埋める
4. ネームを `assets/episodes/0002/` に配置してリンク
5. PR を立ててセルフレビュー
